# Pebble TXT Reader (All Features)

Features included:
- Read TXT from embedded resource (resources/data/book.txt)
- Page-style pagination (Next/Prev pages)
- Dynamic font size adjustment (long Select cycles sizes)
- Theme toggle (short Select toggles light/dark)
- Persist current page, font size and theme across app restarts
- No companion, no external font selection
- Dockerfile + GitHub Actions workflow to build using pebble-tool inside a container

Replace the sample `resources/data/book.txt` with your own .txt file before building.
